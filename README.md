# Java学习项目

## 项目结构

```
src/
├── chapter01_intro/        # 第1章-入门篇（初始Java、JDK、IDEA、HelloWorld、注释）
├── chapter02_syntax/       # 第2章-语法篇（字面量、变量、数据类型、运算符）
├── chapter03_logic/        # 第3章-判断-选择-循环（if、switch、for、while）
├── chapter04_array/        # 第4章-数组
├── chapter05_method/        # 第5章-方法
├── chapter06_principle/     # 第6章-原理篇（Java运行机制、内存分配）
├── chapter07_algorithm/     # 第7章-算法训练
├── chapter08_oop/          # 第8章-面向对象基础
├── chapter09_oop_theory/    # 第9章-面向对象原理篇
├── chapter10_oop_advance/   # 第10章-面向对象进阶（static、final、枚举）
├── chapter11_oop_high/      # 第11章-面向对象高级（继承、多态、抽象类、接口、内部类）
├── chapter12_api/          # 第12章-常见API（String、StringBuilder、ArrayList）
├── chapter13_project/       # 第13章-项目（游戏）
└── chapter14_classes/       # 第14章（Math、System、Object、包装类、BigInteger、BigDecimal）
```

## 使用方法

1. 每学完一章，在对应目录创建练习文件
2. 使用 `javac` 编译，`java` 运行
3. 作业文件命名建议：`Exercise01.java`、`Exercise02.java`...
